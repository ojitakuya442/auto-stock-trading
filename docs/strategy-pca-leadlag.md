# 戦略：日米業種リードラグ × 部分空間正則化PCA

> **出典**: 中川慧, 竹本悠城, 久保健治, 加藤真大「部分空間正則化付き主成分分析を用いた日米業種リードラグ投資戦略」
> 人工知能学会第二種研究会資料 2026年 FIN-036号 pp.76-83 / [DOI](https://doi.org/10.11517/jsaisigtwo.2026.FIN-036_76)

本ドキュメントは、上記論文をベースに**個人投資家10万円スケールで実装可能な戦略**として再構成したもの。

---

## TL;DR

- **戦略**: 米国セクターETFの**当日リターン**から、翌営業日の日本セクターETFの**寄付→引け**リターンを予測 → 上位30%ロング/下位30%ショートの等ウェイト
- **手法**: 部分空間正則化付きPCA（Subspace-Regularized PCA）で安定的な共通ファクター（Global / Spread / Cyclical-Defensive）を抽出
- **論文の実証結果（2010-2025）**:
  - **年率リターン 23.79%**
  - 年率リスク 10.70%
  - **R/R（シャープ的指標）2.22**
  - 最大ドローダウン -9.58%
  - **Fama-French 3 / Carhart 4 ファクターでも α=22.17-22.23%（t=6.69-6.73, 1%有意）**
- **個人投資家への魅力**:
  - 対象が**東証上場の流動性ETF（NEXT FUNDS TOPIX-17）+ 米国Select Sector SPDR**で、誰でもアクセス可能
  - 取引頻度は**日次1回（寄付発注 → 引けクローズ）**、実装が現実的
  - スプレッドや見せ玉リスクが低い

論文オリジナルは「ロング上位30% / ショート下位30%」のロングショート戦略で、信用取引が前提（保証金30万円〜）。
**個人10万円では以下の改変版を採用**:

| バリエーション | 内容 | 期待される性能変化 |
|---|---|---|
| **A. ロングオンリー（推奨）** | 上位30%（=17銘柄中5-6銘柄）を等ウェイトでロング、現金均等配分 | 期待リターンは半減程度、シャープ比率は1.0前後を想定 |

## 2. 論文手法の詳細

### 2.1 入力データ

| 市場 | ETF | 銘柄数 |
|---|---|---|
| **米国** | Select Sector SPDR（XLB, XLC, XLE, XLF, XLI, XLK, XLP, XLRE, XLU, XLV, XLY） | 11 |
| **日本** | NEXT FUNDS TOPIX-17 業種別 ETF（1617.T〜1633.T） | 17 |

**期間**: 2010年1月〜2025年12月（共通エクスポージャー C₀ の構築には2010-2014を使用）

### 2.2 アルゴリズム（要約）

1. **標準化**: ローリングウィンドウ L=60 日で各ETFの close-to-close リターンを z スコア化
2. **共通エクスポージャー V₀ 構築**: 経済構造に基づき、以下の3次元を事前定義
   - v₁ = グローバルファクター: 全ETF同方向 (1, ..., 1)
   - v₂ = スプレッドファクター: 米国正・日本負 (1_NU, -1_NJ)
   - v₃ = シクリカル/ディフェンシブ: 業種感応度の符号付きベクトル
3. **正則化共分散**: `C^reg_t = (1-λ) C_t + λ C₀`、λ=0.9
4. **固有分解**: 上位 K=3 の固有ベクトル V_t を取得、米国側 V_{U,t} と日本側 V_{J,t} に分割
5. **ファクタースコア**: `f_t = V_{U,t}^T z_{U,t}`（米国当日ショックの3共通モード分解）
6. **シグナル**: `ẑ_{J,t+1} = V_{J,t} f_t = B_t z_{U,t}`（日本翌日リターン予測）
7. **ポジション構築**:
   - ロング: 予測リターン上位 q=30%
   - ショート: 予測リターン下位 q=30%
   - 等ウェイト、ネット中立（合計ウェイト=0、合計絶対値=2）

### 2.3 ベースラインとの比較（論文 Table 2）

| 戦略 | 年率リターン | 年率リスク | R/R | MDD |
|---|---|---|---|---|
| MOM（単純モメンタム） | 5.63% | 10.59% | 0.53 | -16.97% |
| PCA PLAIN（正則化なし） | 6.24% | 9.94% | 0.62 | -23.65% |
| **PCA SUB（提案手法）** | **23.79%** | **10.70%** | **2.22** | **-9.58%** |
| DOUBLE（MOM × PCA SUB ダブルソート） | 18.86% | 11.16% | 1.69 | -12.10% |

→ **正則化が決定打**。共分散行列を共通エクスポージャー方向へ縮約することで、低ランク構造を保ちつつノイズを抑制。

## 4. 実装上の注意

### 4.1 米国データの取得タイミング
- 米国市場は日本時間早朝5:00〜6:00頃にクローズ
- → **日本の寄付（9:00）までに前日米国終値を取得**して z_{U,t} を計算
- yfinance は前日終値を翌日午前中に確実に提供

### 4.2 共通エクスポージャー V₀ の再構築
- 論文では「2010-2014」を初期推定期間として使用
- **本実装では拡張ウィンドウ方式**（毎年1月にV₀を再推定）を採用してドリフト対応
- v₃（シクリカル/ディフェンシブ）の符号付与は **業種マッピング表として外部YAML化**（経済学的解釈の修正余地を残す）

### 4.3 単元未満株への対応
- TOPIX-17 ETFは1単元10口（一部100口）。10万円で全17銘柄に分散はできない
- ロングオンリーの上位5銘柄に絞っても1銘柄あたり2万円 = ETF価格次第で1単元しか買えない
- → **kabu Station APIのプチ株（単元未満株）を活用** or **対象銘柄を流動性上位5-7に絞る**

### 4.4 リスクオフ局面の挙動
- 論文によると MDD -9.58% は2010-2025で達成
- ただし**コロナショック（2020年3月）級の局面**ではモデルが追従できない可能性あり
- → §[design.md §7](design.md) のキルスイッチ（1日 -3% で停止）と併用

## 付録：論文の理論的裏付け（Proposition 2 の意味）

論文は次のモデルを置く:

```
z_{U,t}   = V*_U · g_t + ε_{U,t}     ... (米国観測)
z_{J,t+1} = V*_J · g_t + ε_{J,t+1}   ... (日本翌日観測、共通因子 g_t は米国で先に判明)
```

このとき、平均二乗誤差を最小化する z_{J,t+1} の z_{U,t} に対する**最良線形予測（BLP）**は:

```
B* = (1 / (1 + σ²_U)) · V*_J · V*_U^T,   rank(B*) ≤ K
```

→ つまり「日米共通因子に分解して投影する」というナイーブな手順が、**理論的にも最適**であることが保証されている。
正則化付きPCAは V*_U, V*_J の安定推定器として機能する。

これは「とりあえずモメンタムでバックテスト → アタリだったから採用」型の戦略とは異なり、**経済構造の理論モデル → アルゴリズム → 実証**の三段階で強化されている点が信頼性の根拠。
